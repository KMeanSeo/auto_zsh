Run with

```
git clone https://github.com/KMeanSeo/auto_zsh.git && cd zsh-setup && chmod +x auto_zsh.sh && ./auto_zsh.sh
```

Multi user version

```
sudo apt install unzip -y
wget https://github.com/KMeanSeo/auto_zsh/archive/refs/heads/multiuser.zip && unzip multiuser.zip && cd auto_zsh-multiuser
chmod +x auto_zsh.sh
sudo ./auto_zsh.sh
```
